⚔️ PROJECT

CLEAN QUEST

学校の掃除を「義務」から「冒険」へ

青稜中学校・高等学校

3年3組 土蔵 創一

01. CONCEPT

現状の課題 (Problem)

  • 掃除 ＝ 「面倒くさい義務」
  • サボりや雑な掃除が発生しやすい
  • 真面目にやっても評価されない

解決策 (Solution)

「掃除 ＝ ミッション」と再定義。

ゲーミフィケーションを取り入れ、
生徒が楽しみながら参加できるシステムを作る。

02. DEMO

百聞は一見にしかず。実際の画面をご覧ください。

※QRコード読み取りは実機または別ウィンドウ推奨

03. SYSTEM - 1

📱 QRチェックイン

  • 掃除用具入れやエリアにQRコードを設置
  • iPad / スマホで読み取ると「クエスト受注」
  • 「いつ・誰が・どこで」始めたか自動記録

👉 出席確認の手間ゼロ化

03. SYSTEM - 2

🗺️ 汚染エリアの浄化

掃除の進捗をマップ上で可視化


■ 汚染 (掃除前) ➡ ■ 浄化 (完了)

「学校が綺麗になっていく」達成感を可視化

03. SYSTEM - 3

🏆 クラス対抗ランキング

  • 出席率・美化度を XP (経験値) 化
  • クラス対抗で順位を競う
  • 上位クラスには月替わりの報酬を用意
    （例：アイス無料券など）

連帯感を生み、掃除を「チーム戦」に変える

04. IMPACT

導入メリット

  • 生徒: 「やらされる掃除」から「やりたくなるクエスト」へ
  • 学校: 校内環境の劇的な改善
  • 社会: ゴミ分別を通じた環境意識の向上

🌏 SDGsへの貢献

目標4 (教育) / 目標11 (まちづくり) / 目標12 (つくる責任)

CONCLUSION

Project Clean Questは、
退屈な15分間を
「最高の冒険」に変える
次世代の学校改革ツールである。
